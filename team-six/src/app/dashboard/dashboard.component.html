Welcome to the home page of our app.

Our application uses the Twitter API to receive tweets based on a specific #hashtag.

To begin:

  1. Enter a "hashtag" term in the search box. Do not include the # symbol in the search field. Then hit go.
    • The default hashtag is: coffee.
  2. Select a button
    • DISPLAY TWEETS : Will display the most recent tweets.
    • HASHTAGS : Will display other similar hashtags.
    • LOCATION : Will display the location of users.
    • CONNECT : Will display other people to connect with.
